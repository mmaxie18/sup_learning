Prerequisites: Conda installation

1. Set up the conda ml_env using the included requirements.txt file.
>> conda create --name ml_env --file requirements.txt
2. Activate ml_env.
>> conda activate ml_env
3. run python __init__.py within the supervised_learning directory.
>> python __init__.py
